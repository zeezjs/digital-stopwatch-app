A fully functional digital stopwatch built with vanilla JavaScript, HTML, and CSS, designed to demonstrate state management, intervals, and DOM manipulation.

Features Completed
Start, pause, and stop functionality
Time persists when paused and resumes correctly
Formatted display for hours, minutes, seconds, milliseconds
Responsive and polished UI with distinct button colors and hover effects
Smooth and consistent digit display using fixed widths
Planned Features
Lap functionality to record intermediate times
Reset without stopping the stopwatch
Optional UI enhancements such as animations for milliseconds
